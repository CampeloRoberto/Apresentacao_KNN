# KNN — Apresentação Interativa

## Como rodar

Abra um terminal na pasta `Algorítimo` e escolha um dos métodos abaixo.

---

### Python

```bash
python -m http.server 8080
```

---

### Node.js

```bash
npx serve . -p 8080
```

---

### VS Code — Live Server

1. Instale a extensão **Live Server**
2. Clique com botão direito em `Algoritimo KNN.html`
3. Selecione **Open with Live Server**

> O Live Server abre automaticamente no browser na porta correta.

---

## Acessar

Após iniciar o servidor (Python ou Node), abra no browser:

```
http://localhost:8080/Algoritimo%20KNN.html
```
